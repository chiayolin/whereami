whereami
========

Print machine's public IP and its respective geolocation.

Installing
----------

If you have Homebrew installed on your machine then:

```sh
$ brew tap chiayolin/cask
$ brew install whereami
```

Or you can just run the executable in `bin/`:

```sh 
$ bin/whereami
```

Usage
-----

Simply:

```sh
$ whereami
```
